# Custom Fonts Directory

Place your custom font files here:

## Primary Font: ANCORLI
- `ancorli-regular.woff2` (preferred)
- `ancorli-regular.woff`
- `ancorli-regular.ttf`

## Secondary Font: TECHNO NUE
- `techno-nue-regular.woff2` (preferred)
- `techno-nue-regular.woff`
- `techno-nue-regular.ttf`

## Notes
- WOFF2 format is preferred for best compression
- The application will fall back to Geist fonts if custom fonts are not found
- Fonts are already configured in `app/globals.css`
